* BufferWriter은 << Operator을 오버로딩하여 SendBufferChunk에 << 피연산자 형식에 맞게 데이터를 쓸 수 있다.
* BufferReader은 >> Operator을 오버로딩하여 recvBuffer로 부터 >> 피연산자 형시겡 맞게 데이터를 읽어 올 수 있다.
